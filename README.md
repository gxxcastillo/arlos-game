# Arlo's 3 legged cat

This is a fun little game that Arlo is making lo learn how to program.

## Dependencies

1. [Node.js](https://nodejs.org/en) - recommended installation via [NVM](https://github.com/nvm-sh/nvm)
2. [PNPM](https://pnpm.io/)

## Setup

1. Clone this repo
2. `cd` into project
3. run `pnpm install`
4. run `pnpm run serve`
